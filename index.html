Chinese Training 설치 가이드

이 프로그램은 Python으로 만들어진 중국어 단어 학습 프로그램입니다.

아래 순서대로 설치하면 정상적으로 실행할 수 있습니다.

1. Python 설치

먼저 Python이 설치되어 있어야 합니다.

공식 사이트: https://www.python.org/downloads/

Windows

  1. 위 사이트에서 Python 설치 파일 다운로드
  2. 설치 실행
  3. "Add Python to PATH" 체크 후 설치

Mac

  1. 위 사이트에서 macOS용 Python 다운로드
  2. 설치 파일 실행 후 설치 진행
  3. 설치 후 터미널에서 아래 명령어 입력

python3 --version

버전이 출력되면 정상 설치된 것입니다.

2. GitHub에서 파일 다운로드

GitHub 저장소: chineseTraining

  1. 위 링크 접속
  2. 초록색 Code 버튼 클릭
  3. Download ZIP 클릭
  4. 다운로드된 ZIP 파일 압축 해제

3. customtkinter 설치

프로그램 실행에 필요한 라이브러리인 customtkinter를 설치해야 합니다.

Windows

압축 해제한 폴더 위치에서 터미널 또는 CMD 실행 후:

pip install customtkinter

Mac

터미널 실행 후:

pip3 install customtkinter

주의:
Mac에서는 보통 pip 대신 pip3를 사용해야 합니다.

4. 프로그램 실행

다운로드한 폴더 안에서 problemMaker_gui.py 파일을 실행하면 됩니다.

Windows

  1. 압축 해제한 폴더 열기
  2. 실행할 Python 파일 확인
  3. 터미널에서 아래 명령어 실행

python problemMaker_gui.py

Mac

  1. 터미널 열기
  2. 프로그램 폴더로 이동
  3. 아래 명령어 실행

python3 problemMaker_gui.py

문제가 발생한다면

  • python 명령어가 안 되면 Python이 제대로 설치되지 않았을 수 있습니다.
  • ModuleNotFoundError가 뜨면 customtkinter 설치를 다시 진행하세요.
  • Mac에서는 대부분 python3, pip3를 사용해야 합니다.
